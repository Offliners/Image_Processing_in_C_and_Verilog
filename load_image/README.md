# Load Image
Read image from BMP (bitmap) file , and then write it into another.

| Input                   | Output                  |
| ----------------------- | ----------------------- |
| ![input](./lena256.bmp) | ![output](./output.bmp) |

## Usage
```shell
# C
$ cd ./load_image/C
$ make
$ ./load_image.o ../lena256.bmp

# Verilog
```
